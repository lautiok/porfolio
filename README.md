# ¡Bienvenido a mi Portafolio!

🚀 Este es mi sitio web basado en ReactJS, donde puedes explorar los proyectos que he desarrollado, obtener información de contacto y descubrir mis habilidades técnicas. 💻

---

## Características

- 💡 Experiencia fluida y estable con **ReactJS**.
- 🎨 Diseño intuitivo para una navegación suave.
- 🖌️ Presentación impactante de mis proyectos.

---

## Tecnologías Utilizadas

- ⚛️ **ReactJS:** Desarrollé este sitio web utilizando ReactJS para una interfaz de usuario dinámica y eficiente.
- 🎨 **Sass:** Utilicé Sass para una gestión de estilos más efectiva, facilitando el mantenimiento y la escalabilidad del diseño.
- 📜 **JavaScript:** El lenguaje de programación principal para la lógica del lado del cliente, mejorando la interactividad y la experiencia del usuario.
- 🅱️ **Bootstrap:** Se empleó Bootstrap para un diseño rápido y responsivo, garantizando una apariencia consistente en diferentes dispositivos.

---

## Funcionalidad

- 🌐 **Navegación React-Router:** La navegación del sitio se realiza de manera dinámica y suave gracias a React-Router, proporcionando una experiencia de usuario sin interrupciones.
- 🎉 **Animaciones Atractivas:** Se han incorporado animaciones de [Animate.css](https://animate.style/) y [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) para dar vida a la interfaz y crear transiciones llamativas.
- 🚀 **Proyectos desde Firebase:** Los proyectos son cargados dinámicamente desde Firebase, utilizando la API correspondiente. La información se presenta de manera clara y atractiva.
- 📈 **Exploración de Proyectos:** En la página de inicio, se muestran tres proyectos destacados. Los usuarios pueden explorar más proyectos haciendo clic en "Ver más proyectos", lo que revela una lista completa de todos los proyectos en la sección de portafolio.

---

## Instalación de React

Si deseas ejecutar este proyecto en tu máquina local, puedes seguir estos pasos:

1. **Clona el Repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/tu-proyecto.git
   cd tu-proyecto
   ```
2. **Instala las Dependencias:**
   ```bash
   npm install
   ```
3. **Inicia la Aplicación:**
   ```bash
   npm start
   ```
4. Abre tu navegador y visita http://localhost:3000 para ver la aplicación.
